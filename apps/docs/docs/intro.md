# JES

JES (Joe Ecosystem System) is an open-source platform for monitoring, controlling, and automating developer workstations from anywhere.

## Features

- System Monitoring
- Docker Monitoring
- Remote Commands
- Device Pairing
- Mobile Application
- Self Hosted

## Architecture

Flutter App
↓
JES Cloud API
↓
JES Agent

## Current Status

Version: 0.1.0-alpha

JES is currently under active development.
